CSS is a stylesheet language that is used to style elements in an HTML document.

Border

All block-level elements have a border, which by default is not visible. This property is used to style such border, and it takes one through three values: width, style, color.

border: medium solid rgb(17, 74, 110);

Border-side

There are four more specific border properties: border-left, border-right, border-top, border-bottom. These allow more specific border manipulation.

border-right: medium solid rgb(17, 74, 110);
border-top: dashed rgb(30, 142, 212);
border-left: double medium rgb(17, 74, 110);
border-bottom: solid;

Border Radius

Border radius defines the radius of the border's corners, allowing for round corners.

border: medium solid rgb(17, 74, 110);
border-radius: 10px;

Background Image

Sets the background for an element with a url or a path to an image.

Example

Color

The color property sets the color of the text in that element. It can receive a name of a color, an rgb value, or a hex value.

color: #da6c12;

Font

Font is a short-hand property that can receive the following values: style, variant, weight, size, family.

font: italic small-caps bold 12px "Comic Sans MS", "Comic Sans";

Font Family

Specifies the font for an element

font-family: "Comic Sans MS", "Comic Sans";

Font Style & Weight

The font style allows the font to be either normal or italic, while the weight can be bold, or a more specific number for it's width.

font-style: italic;
font-weight: bold;

Width and Height

The width and height properties allow to set the size of an element. They can be the desired value, or a percentage of the containing block.

width: 30%;
height: 75px;
background-color: rgb(17, 74, 110);

List Style

The list-style property allows to define the following values for a list: type, image, and position.

  • list-style: square inside;
  • list-style: decimal inside;

Margin

Margin is the space between the border of the block, and the rest of the elements. Changing this value can help move content around to make it more readable.

margin-left: 100px;

Padding

Padding is similar to the margin, but is the space between the content of the block, and the border of the block. So changing this value also moves the content, but dodes not move the block.

padding-left: 100px;

Overflow

The overflow property controls how text behaves when it goes outside the limit of the container. The most used options are: visible, hidden, scroll, and auto (like scroll but adds a scrollbar only when necessary).

color: white;
background-color: rgb(17, 74, 110);
width: fit-content;
height: 30px;
overflow: scroll;

Display

The display property controls how content within an element is rendered. As such, it can be a very helpful tool to control the layout of a page

display: block;
margin-left: auto;
margin-right: auto;

Position

The position property controls how elements are positioned. There are very helpful values here, like fixed and sticky which changes or maintains the position of that element when things are scrolled. The static value is the default. With relative, positioning starts in its original spot, but we can change from there

position: relative;
left: 30px;

Filter

Filter applies visual effects to elements. There are several options like greyscale, or blur.

filter: blur(1px);
