Tarea 1: HTML Cheat Sheet

HTML, HyperText Markup Language, gives content structure and meaning by defining that content as, for example, headings, paragraphs, or images.

<a>

The anchor element is used to link from a web page to another. This is done by using the href attribute. With HTML5, anchors can wrap other elements to make blocks of content links. To open links in a new window: target=”_blank”. We can also link to parts of the same page using ids, as such: href=”#top”.

For example, the following element renders as such:
<a href="[url]" target="_blank">Other Page</a>
