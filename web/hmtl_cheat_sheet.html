HTML, HyperText Markup Language, gives content structure and meaning by defining that content as, for example, headings, paragraphs, or images.

<a>

The anchor element is used to link from a web page to another. This is done by using the href attribute. With HTML5, anchors can wrap other elements to make blocks of content links. To open links in a new window: target=”_blank”. We can also link to parts of the same page using ids, as such: href=”#top”.

For example, the following element renders as such:
<a href="[url]" target="_blank">Other Page</a>
Other Page

<p>

The paragraph element is a block-level element that semantically represents, well, a paragraph.

Any content within it will be it's own paragraph, so it's used like this:
<p>
    Paragraph content
</p>

<h1>, <h2>, ..., <h6>

The heading element is a block-level element that helps establish hierarchy. <h1> should be the heading of a page or section.

Example of the three bottom heading element:

<h4>

<h5>
<h5>

<div>

The div element is a block-level commonly used to identify large groupings of content.

It's used like so:
<div>
    Content that will be it's own block.
</div>

<span>

The span element is a inline-level commonly used to identify smaller groupings of text.

As a inline element it can be used to group a small section of text. For example:
You can make a word <span style="color:cornflowerblue">blue</span>
You can make a word blue

<strong>

The strong element is used to make text bold and place a strong importance on it. It is different from <b> semantically.

It wraps around the word that you want to mark as important:
This word is <strong>important!</strong>
This word is important!

<i>

The italicized element is an inline-level that is used to define an alternate tone within a text section. Only used when there is no other more appropiate semantic element.

It is used like so:
Like I always say, <i>carpe diem</i>.
Like I always say, carpe diem.

<u>

The underline element is an inline-level that underlines the text selected. It is usually used to mark misspelled words.

For example:
This word is <u>mispeled</u>.
This word is mispeled.

<b>

The bold element is an inline-level that makes the text bold. It should be used only when neither a heading, strong, or em element makes sense.

For example:
This word is <b>bold</b>.
This word is bold.

<s>

The strike element is an inline-level that renders text with a strikethrough. It should be used for text that is no longer accurate.

For example:
<s>Low availability!</s> <strong>SOLD OUT!</strong>
Low availability! SOLD OUT!

<small>

The small element is an inline-level that defines smaller text. It's usually used for text like copyright or asides.

For example:
MyBusiness <small>Reserved Rights.</small>
MyBusiness Reserved Rights.

<em>

The emphasis element is an inline-level used for text that should be emphasized. It italicizes the text, but it's different semantically from <i>

For example:
This <em>cannot</em> be happening.
This cannot be happening.

<header>

The header is a block-level element used to identify the top of a page.

It usually has a heading, and some paragraphs. For example, this page's header is:

<header>
    <h1>Tarea 1: HTML Cheat Sheet</h1>
    <a href="../index.html">Go back home</a>
    <p>
        <em>HTML</em>, HyperText Markup Language, gives content structure and meaning by defining that content as, for example, headings, paragraphs, or images.
    </p>
</header>

<section>

The section block-level element is used to identify a thematic grouping of content. Can include a heading. Break up and provide hierarchy to a page.

It's used as any block-level element:
<section>
    Content and elements within this section.
</section>

<article>

The article block-level element is used to identify a section of independent, self-contained content. Often used when the content could be redistributed elsewhere.

It's used as any block-level element:
<article>
    Content and elements within this article.
</article>

<aside>

The aside block-level element holds content such as sidebars, or brief explanations, related to the content surrounding it.

It's used as any block-level element:
<aside>
    Content and elements within this aside.
</aside>

<footer>

The footer block-level element identifies the closing or end of a page, article, section, or other segment.

It's used as any block-level element:
<footer>
    Content and elements within this footer.
</footer>

<nav>

The nav element identifies a section of major navigational links on a page. Links within nav will link to other pages or parts within the same website.

It's used as any block-level element:
<nav>
    Content and elements within this nav.
</nav>

<img>

The image inline-level element is used to embed images in an HTML page. There are two important attributes: src for the path of the image, and alt for if the image cannot be displayed.

For example, the following tag would render the following image:

<img src="https://hips.hearstapps.com/hmg-prod/images/dog-puppy-on-garden-royalty-free-image-1586966191.jpg?crop=1.00xw:0.669xh;0,0.190xh&resize=1200:*" alt="Cute dog" height="200">

<br>

The break inline-level element forces a line break.

For example:
The break element <br> forces a line break.

The break element
forces a line break.

<hr>
